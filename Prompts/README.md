## Prompt File Format

Each prompt file is a simple markdown document:

```markdown
First prompt describing a scene or element
Second prompt with a variation of lighting
Third prompt from different camera angle
Fourth prompt with different time of day
```

**Key formatting rules**:
- One prompt per line
- NO blank lines between prompts
- Each line is a complete, standalone prompt
- Each file represents a specific concept or category

## Using These Prompts

### For Image Generation (Midjourney, DALL-E, Stable Diffusion, Freepik, etc.)
- Each line is a complete prompt ready to use
- Select prompts based on needed section, style, or element
- Combine prompts from different files for variety
- Iterate using variations and different parameters

### For Video Generation (Runway, Pika, Sora, Kling, etc.)
- Use prompts from transition/motion categories
- Describe camera movement and temporal changes
- Never include duration numbers in prompts
- Focus on smooth, flowing movements

### For 3D Scene Building
- Use prompts as concept references for modeling
- Extract color palettes from prompt descriptions
- Identify key elements to model and arrange
- Use lighting descriptions to set up 3D scene lighting
